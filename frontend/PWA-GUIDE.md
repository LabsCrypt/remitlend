## 🚀 Deployment Instructions

### 1. Environment Setup
```bash
# Set production environment
export NODE_ENV=production

# Build the application
npm run build
```

### 2. Required Environment Variables
```bash
# PWA Configuration
NEXT_PUBLIC_APP_URL=https://your-domain.com
NEXT_PUBLIC_APP_NAME=RemitLend
```

### 3. HTTPS Requirements
PWA requires HTTPS in production:
- ✅ Use SSL certificate (Let's Encrypt recommended)
- ✅ Configure HTTPS redirect
- ✅ Update all API endpoints to HTTPS
- ✅ Test with production domain

### 4. Service Worker Deployment
The service worker is automatically generated by `next-pwa`:
- ✅ Located at `/public/sw.js` after build
- ✅ Automatically registered on app load
- ✅ Updates on new app versions

### 5. Icon Generation (Required)
```bash
# Install ImageMagick (if not installed)
brew install imagemagick  # macOS
sudo apt-get install imagemagick  # Ubuntu

# Generate all required icons
./generate-icons.sh

# Verify icons exist
ls -la public/icons/*.png
```

### 6. Manifest Verification
After deployment, verify:
- ✅ `https://your-domain.com/manifest.json` is accessible
- ✅ All icon paths are correct
- ✅ Start URL points to correct location
- ✅ Display mode is "standalone"

## 🔧 Troubleshooting

### Common Issues

#### Install Prompt Not Showing
```javascript
// Check in browser console
navigator.standalone // iOS
window.matchMedia('(display-mode: standalone)') // Android
```

**Solutions:**
- Ensure HTTPS is enabled
- Check user engagement criteria (requires user interaction)
- Verify manifest.json is accessible
- Clear browser cache and retry

#### Service Worker Not Registering
```javascript
// Check in DevTools > Application > Service Workers
navigator.serviceWorker.getRegistrations()
```

**Solutions:**
- Verify `/sw.js` is accessible
- Check for HTTPS requirement
- Clear service worker cache
- Restart browser

#### Offline Page Not Working
**Solutions:**
- Verify offline.html exists
- Check service worker cache
- Test with actual network disconnection
- Verify fallback configuration

#### Icons Not Loading
**Solutions:**
- Generate PNG icons using provided script
- Verify all required sizes exist
- Check file paths in manifest.json
- Test with different devices
